# 更新日志

## [2.0.2] - 2026-04-25

### UI 现代化 (Fluent Design 风格)

#### 视觉主题全面升级
- 采用 Windows 11 Fluent Design 设计语言重写 QSS 主题
- 全局 8px 圆角、卡片式 GroupBox (左侧蓝色色条 + 轻阴影)
- 输入控件 Fluent TextBox 风格：底部 2px 重点线，聚焦时变为蓝色
- 按钮区分主操作 (蓝色 Accent)、停止 (红色 Danger)、清空 (透明次要) 三种样式
- 侧边栏 WinUI NavigationView 风格，选中项白色背景 + 蓝色文字

#### 侧边栏药丸形指示器
- 新增 `_PillIndicator` 组件：4px 宽蓝色圆角竖条，随选中项平滑滑动
- 使用 `QPropertyAnimation` 实现丝滑的 Y 轴位移过渡

#### 页面切换动画
- 新增 `AnimatedStackedWidget` 组件：切换功能页面时带水平滑动 + 淡入过渡
- 左右方向自适应（向前/向后切换方向不同）
- 使用 `QParallelAnimationGroup` 并行运行滑动 + 透明度动画

#### 进度仪表盘
- 新增 `ProgressDashboard`：4 个指标卡片（进度、速度、帧、剩余时间）+ 渐变色进度条
- 新增 `FFmpegProgressParser`：实时解析 FFmpeg stderr 输出提取 frame/fps/time/speed 等指标
- 任务未运行时仪表盘自动隐藏，任务开始时显示不确定进度，获取数据后实时更新

#### 日志面板美化
- 日志面板改为深色终端风格 (`#1e1e1e` 背景，Cascadia Code 字体)
- ANSI 颜色映射调整为深色背景适配的高对比度色值
- 新增平滑滚动效果，使用 `QPropertyAnimation` 缓动到底部

#### 按钮与状态栏动画
- 执行按钮点击时带微缩弹回动画 (`button_press_anim`)
- 任务完成/失败时状态栏颜色短暂闪烁反馈 (`status_flash`)

### Bug 修复
- 修复日志面板滚动动画使用 `DeleteWhenStopped` 导致悬空 C++ 引用，压制开始后崩溃的问题
- 修复 QSS 中 `font-size: 0px` 导致的 `QFont::setPixelSize: Pixel size <= 0` 警告
- 日志面板默认高度加高，分割器比例调整，日志区域更加宽敞

### 新增文件
- `src/ui/animations.py` — 可复用动画工具集 (淡入、页面滑动、按钮脉冲、状态闪烁)
- `src/ui/ffmpeg_progress.py` — FFmpeg 输出进度解析器
- `src/ui/progress_dashboard.py` — 进度仪表盘 UI 组件

## [2.0.0] - 2026-04-25

### 重大更新: GUI 框架迁移 Gooey → PyQt6

本版本将图形界面框架从 Gooey (wxPython) 完全迁移至 **PyQt6**，实现了更灵活的 UI 布局、动态控件联动和异步任务管理。

#### 新架构
- 新增 `src/ui/` 模块：`main_window.py` 主窗口、`sidebar.py` 侧边栏、`log_panel.py` 日志面板、`task_runner.py` 异步任务
- 新增 `src/ui/tabs/` 模块：12 个独立功能标签页，各自包含 `build_args()` 方法构建兼容参数
- 新增 `src/ui/base_tab.py`：通用控件创建辅助方法基类
- 新增 `src/ui/args_builder.py`：`ArgsNamespace` 兼容 `argparse.Namespace`，后端执行器无需修改
- 新增 `src/ui/theme.py`：QSS 统一主题样式

#### 封装转换自动识别轨道
- 选择输入文件后自动调用 `ffprobe` 探测音频/字幕轨道
- 以复选框形式展示每条轨道的编解码器、声道、语言、码率等详情
- 用户直接勾选/取消即可选择保留哪些轨道，无需手动查看编号
- 批量模式下自动显示第一个文件的轨道信息

#### 容器兼容性警告
- 封装转换时自动检测源文件编码与目标容器的兼容性
- 不兼容时在界面中显示警告并给出建议 (如 "VP9 不被 MP4 容器支持，建议改用 MKV")
- 覆盖 MP4、WEBM、MOV、AVI、TS 等主流容器的兼容性规则

#### 动态控件联动
- 视频压制：非自定义预设时自动禁用编码器/质量参数编辑；NVENC 档位仅在选择 NVENC 编码器时可用
- 视频压制：音轨/字幕自定义编号输入框仅在选择「自定义选择」时启用
- 视频压制：分发目录仅在分发模式非「不分发」时可编辑
- 封装转换：自定义后缀名输入框仅在选择「自定义」预设时启用
- 图片转换：自定义格式输入框仅在选择「自定义」时启用；质量滑块仅对 JPG/WEBP 有效
- 素材质量检测：自定义分辨率输入框仅在选择「自定义」时启用
- 批量重命名：输出路径仅在复制/移动模式时可编辑
- 露骨图片识别：打码相关控件仅在启用打码时可操作；自定义图片仅在选择自定义模式时启用

#### 内联帮助提示
- 新增 `add_hint()` 方法，支持信息 (蓝)、警告 (橙)、提示 (绿) 三种样式
- 所有功能页面增加来自 `gui_tabs.py` 的关键操作说明，直接可见而非仅作为 tooltip
- 使用说明页面重写为实时渲染的 HTML 帮助文档

#### 通用增强
- 所有文件输入框支持拖放 (FileDropLineEdit)
- ANSI 彩色日志渲染支持
- 版本号改为从 git tag 自动读取，CI 构建时注入 (`src/_version.py`)
- 更新 GitHub Actions 工作流以适配新版本注入方式

### 依赖变更
- 移除: `Gooey>=1.0.8`
- 新增: `PyQt6>=6.5.0`

### 视频压制增强

#### 输出格式选择
- 新增输出容器格式选项：MP4 (默认)、MKV、MOV、与输入相同、自定义
- 自定义模式支持手动输入任意扩展名 (如 ts、webm)
- 输出格式影响自动生成的输出路径扩展名

#### 更多质量预设
- 新增「较小体积」x264 CRF22 预设，适合对画质要求不高、需要快速出片的场景
- 新增「均衡画质」x265/HEVC CRF20 预设，同画质下体积约为 x264 的 60-70%
- 重命名现有预设以更清晰地标注编码器和参数

#### 音频/字幕轨道自动检测
- 视频压制模块选择输入文件后自动调用 `ffprobe` 探测音频/字幕轨道
- 以复选框形式展示每条轨道的详细信息 (编解码器、声道、语言、码率)
- 音频轨道默认全选，字幕轨道默认不选
- 有探测数据时自动隐藏静态下拉选择，无探测数据时自动回退至静态模式

#### 视频输出说明
- 视频输出区域新增提示：留空分辨率和帧率时保持原始值，不做缩放或变速处理

### CI/CD 增强

#### Release Notes 动态生成
- GitHub Release 说明不再使用静态文本，自动从 CHANGELOG.md 解析当前版本的更新日志
- 同时适用于正式版 (build.yml) 和预发布版 (build-beta.yml) 的工作流
- 构建矩阵提供两版本：标准版、Shield 增强版

## [1.9.0] - 2026-04-13

### 新增功能

#### 媒体元数据检测
- 新增「**媒体元数据检测**」独立功能标签页
- 通过 `ffprobe` 探测视频、音频、图片的完整流信息
- 报告中标注每条音频流和字幕流的编号（`#0`, `#1`...）及语言标签（`jpn`/`chi`...）
- 多轨道文件底部自动附带使用提示，引导用户在压制/封装转换中精确引用
- 支持单文件/多文件选择和目录扫描（可选递归子目录）两种输入模式
- 支持将报告输出为 TXT 文件

#### 音轨/字幕精确选择（视频压制 & 封装转换）
- 「视频压制」新增音轨选择：快捷选择（全部/仅第一条/不保留）+ 自定义编号输入
- 「视频压制」新增字幕流选择：支持保留内封字幕流，可精确指定保留对应编号
- 「封装转换」新增流选择分组，音轨和字幕流均支持快捷模式和自定义模式
- 「封装转换」新增封装转换目标格式用户自定义输入
- 精确编号输入支持多条逗号分隔，如 `0,2` 表示保留 `#0` 和 `#2` 两条流
- 使用 FFmpeg 的可选映射语法（`0:a:N?`），当源文件对应流不存在时不报错

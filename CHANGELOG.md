## [1.8.0] - 2026-03-17

### 音视频抽取功能增强
- **跳过同类型图片转换**: 自动检测输出目录中是否已存在同类型文件（如 .jpg），存在则跳过转换，避免重复工作
- **支持格式扩展**: 支持提取 MP4、MKV、MOV 等视频容器中的音频和视频流
- **多格式支持**: 支持提取 AAC、MP3、AC3、FLAC、EAC3、Opus 等音频格式
---

## [1.7.0] - 2026-02-19

### 批量重命名增强
- **排序规则自定义**: 新增排序设置区块，可选择按文件名或文件大小排序
- **排序方向控制**: 支持升序（从小到大）和降序（从大到小）
- **关键词提前排序**: 新增关键词提前功能，文件名包含特定关键词（如“主视觉图”）的文件将优先排列在序列首部

### 压制后成品分发
- **自动化分发**: 新增压制结束后自动分发功能，可将成品文件复制或移动到指定目录
- **挂载支持**: 配合 WebDAV、SMB 等网络位置挂载，可实现自动上传成品到 NAS 或云端

### GUI 体验优化
- **拖拽支持说明**: 输入框提示语更新，明确支持将文件或文件夹直接拖动到路径输入框内

## [1.5.1] - 2026-01-11

### Shield 功能增强
- **打码范围扩展**: 新增 `expand_pixels` 参数，可向外扩展打码区域 (0-200 像素)
- **多种打码模式**: 马赛克、模糊、黑色遮盖、表情覆盖、自定义图片
- **处理强度通用化**: 处理强度/大小参数对所有模式生效
  - 马赛克: 方块大小
  - 模糊: 半径
  - 黑色遮盖: 圆角大小
  - 表情: 元素比例
- **评级修复**: 添加 `r15`/`r18` 评级映射，修复 unknown 状态
- **报告优化**: 统计 → 详情 → 处理结果，三段式结构
- **RGBA 兼容**: 自动转换 RGBA 为 RGB，修复 JPEG 保存错误

---

## [1.5.0] - 2026-01-11

### 新增功能
- **露骨图片识别 (Shield)**: 基于 AI 的 NSFW 内容检测
  - 识别 B 站投稿过审风险图片
  - 支持 4 级风险分类: safe/sensitive/questionable/explicit
  - 可自定义风险阈值
  - 支持自动马赛克/模糊打码
  - 生成详细检测报告
- **双版本构建**: GitHub Actions 自动构建两个版本
  - 标准版: 体积较小，不含 AI 模块
  - Shield 增强版: 包含露骨图片识别功能

### 技术改进
- 新增 `src/nsfw_detect.py` 核心检测模块
- 使用条件导入实现功能可选
- Matrix 策略并行构建双版本

### 依赖更新 (Shield 版)
- dghs-imgutils >= 0.8.0
- onnxruntime >= 1.15.0

---

## [1.4.0] - 2026-01-11

### 新增功能
- **Docs**:  在顶部菜单栏添加“帮助文档”跳转菜单；在各功能页**底部**添加独立的“在线文档”复制框。
- **字幕兼容模式**: AviSynth + VSFilter 字幕渲染
  - 解决 Windows 系统字体无法正确加载、字重无法匹配等问题
  - 便携版设计，无需安装 AviSynth，解压即用
  - 复用所有现有编码参数
- **NVENC 速度档位控制**: 新增 p1-p7 档位选择
  - 预设模式和自定义模式均可调节
  - 选择"使用预设默认"则使用预设定义的档位
- **编码参数重构**: 使用 `EncodeParams` 数据类统一管理
  - 输入验证 (文件存在、CRF 范围、分辨率格式)
  - 更清晰的代码结构

### 改进
- **默认音频编码**: 改为"复制 (不重新编码)"
- **预设配置完善**: 显式添加 `audio_encoder` 字段
- **测试覆盖增强**: 39 个单元测试用例

### 依赖更新
- 新增 pytest (测试框架)

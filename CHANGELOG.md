# 更新日志 (Changelog)


## [1.2.3] - 2026-01-11

### 新增功能
- **NVIDIA 显卡加速 (HQ) 预设**: 新增画质优先预设，使用 CQ 19 + P7 + 动态码率 VBR。
- **Debug 模式**: 新增 "Debug 模式" 选项，仅输出 FFmpeg 命令到日志和终端，不实际执行压制。
- **日志增强**: 所有的 FFmpeg 执行命令现在都会记录到 `xiaoxue_toolbox.log`。

### 改进与修复
- **Strict Mode (严格模式)**: 非自定义模式下，GUI 中的质量、分辨率、帧率、音频等设置现在会被强制忽略，严格遵循预设。
- **NVENC 参数修复**: 修复了 NVENC 预设下 `cq` 参数可能丢失的问题。
- **预设优化**: 为所有 NVENC 预设增加了 `-rc vbr -b:v 0 -maxrate 20M` 以确保复杂场景下的画质和稳定性。

## [1.1.0] - 2026-01-10

### 新增功能
- 图片格式转换: 批量转换图片格式 (PNG/JPG/WEBP/BMP/GIF/TIFF)
- 批量创建文件夹: 从 TXT 文件读取名称批量创建文件夹，支持自动序号
- 批量序列重命名: 递归/非递归模式批量重命名图片和视频文件
- 封装转换批量模式: 支持多文件批量转换容器格式
- 覆盖原文件选项: 封装转换和图片转换新增覆盖原文件选项
- 通知设置: 支持飞书通知和自定义 Webhook
- 任务完成自动通知: 其他任务完成后自动发送通知

### 改进
- 通知配置回显: GUI 界面现在会显示已保存的通知配置
- 通知配置管理: 新增删除已保存配置的选项
- 优化配置加载状态显示，方便调试
- "质量检测"更名为"素材质量检测"
- 质量检测支持自定义不兼容格式规则
- 封装转换新增更多预设 (MXF, TS, WEBM 等)

### 依赖更新
- 新增 Pillow (图片处理)
- 新增 chardet (编码检测)
